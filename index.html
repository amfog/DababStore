03 · Launch Roadmap

THE PLAN

Four phases from zero to a live, growing shop — using your stream as the engine.

01
// WEEK 1–2 · FOUNDATION
BUILD THE BRAND
  • Lock in Dabab Store name, logo & color palette — dark, misty, neon green/blue
  • Create social handles: @DababStore on TikTok, Instagram, Twitter/X
  • Design 2–3 hero products via Printify or local print shop
  • Set up Fourthwall (free for streamers) or Shopify store with Dabab theme
02
// WEEK 3–4 · HYPE BUILD
TEASE THE DROP
  • Show product design BTS on stream — let chat feel involved
  • Use your Dabab mousepad on stream, let chat ask about it organically
  • "Founding Member" waitlist — first 50 get an exclusive discount code
  • Post hype content 3x/week across all socials
03
// MONTH 2 · LAUNCH
GO LIVE
  • Launch stream: unbox your own products live with chat watching
  • Limited first drop — 50 units per product to create scarcity and hype
  • Pin shop link in all bios; add to stream overlay as lower-third graphic
  • Collab with 1–2 other Wild Rift streamers for a cross-drop
04
// MONTH 3–6 · SCALE
EXPAND THE FOG
  • Launch digital overlay packs for 90%+ margin passive income
  • Monthly "Dabab Drop" — limited seasonal items that sell out fast
  • Build email list via post-purchase flows for direct communication
  • Affiliate program — let other streamers earn % on referrals

05 · Marketing Strategy

HOW TO GET CUSTOMERS

You already have an audience that trusts you. Here's how to convert them.

🔴 Live Stream Tactics
  • Use it on stream. Mousepad, hoodie, mug — all visible on cam. Chat will ask.
  • Milestone drops. "When we hit 100 subs, new Dabab item drops."
  • Buy alerts. Sound + chat shoutout every time someone purchases.
  • Unboxing events. New drop reveals become must-watch moments.
📱 Social Media
  • TikTok/Reels. Setup tours and unboxings convert extremely well.
  • Twitter/X. Announce drops, post gameplay clips, hype the community.
  • BTS content. Show the design process — fans love feeling involved.
  • Wild Rift hashtags. Tap the broader game community beyond your own audience.
🤝 Collabs & Community
  • Collab drops. Partner with another Wild Rift streamer for co-branded items.
  • Gift top supporters. They become walking ads and brand ambassadors.
  • Discord channel. Exclusive deals and early access for server members.
🛒 Shop Setup
  • Fourthwall — Free platform built specifically for streamers. Start here.
  • Bundle deals. "The Dabab Setup": mousepad + mug + overlay pack.
  • Email capture. 10% off first order for signup. Own your audience.
